工程文件说明：
HAREWARE -- 存放外设的驱动程序
OBJ      -- 存放工程生成的hex文件
USER     -- 存放工程路径
使用方法说明：
一.硬件连接
1.本实验需要用到两个usb接口，一个连接单片机用于下载调试代码且为为其供电，
  另一个为指纹模块供电，需要指纹模块和单片机共地。
2.因为lcd的数据端口和led用到相同的io引脚，j11(具体是哪个看一下)跳线帽连接
  off;
  因为继电器和蜂鸣器用到相同的io引脚，j11()跳线帽连接SPEK;
3.具体连线如下：
  (1)指纹模块的VCC通过usb转串口的5V电脑相连，VUSB和GND与单片机的5V和地相连，
  然后单片机的GND通过usb转串口的5V电脑相连。
  (2)指纹模块的RX和TX分别与单片机串口2的TX和RX相连。
  VTOCH和单片机的P2.5相连。连接方式均通过母对母杜邦线
二.操作说明
1.按上述操作连线后，打开单片机的电源按键，可听到蜂鸣器响一下说明单片机和指纹
  模块握手成功，然后能看到lcd上显示初始化成功随后进入空闲状态。
2.通过按键1可设置当前的状态为空闲状态，识别状态，管理员密码输入状态，添加指纹状态
  和删除指纹状态。
3.空闲态下显示当前的日期初始日期为04月16日12时00分00秒，并且日期可调。按下按键2可以选择
  调节日期的日时分，当lcd右上显示d,h,m分别代表日时分。按键3可以使对应的数字加1。按下按键1进入在指纹识别状态。在识别
  状态按下指纹，如果听到连续的三下蜂鸣器的声音说明没有识别到指纹，如果听到一声蜂鸣器的响声
  说明识别到指纹，并且在lcd显示相应的编号。按一下按键1进入管理员密码输入状态，如果通过矩阵
  按键输入了正确密码进入就会指纹删除状态，反之则返回空闲状态。管理员密码在一次开机只需输入
  一次，默认密码为134。在删除指纹状态下会显示指纹的编号和clear_all，通过按键2切换，在clear_all
  模式下按下按键3，会删除所有的指纹，如果成功删除，会在lcd上显示一下OK！暂时没加入删除指定指纹
  的功能。在按一下按键1会进入指纹添加状态，在指纹添加状态下会显示指纹的编号，按下按键2可以在不同的
  编号下切换，共有三个编号分别为id_000,id_001和id_002。在当前编号下输入指纹并按下按键3，如果输入成功
  可以听到蜂鸣器响一下，并在lcd上短暂显示OK！。
